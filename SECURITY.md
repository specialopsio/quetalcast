## Reporting a Vulnerability

If you discover a security vulnerability in QueTal Cast, please report it responsibly.

**Please do not report security vulnerabilities through public GitHub issues.**

Instead, please contact us directly:

- **Email**: [SpecialOPS](https://specialops.io) â€” use the contact form or email on the website
- **Responsible disclosure**: We ask that you give us time to address the issue before any public disclosure

### What to report

- Authentication bypass or session hijacking
- Exposure of sensitive data (credentials, tokens, session data)
- Remote code execution or injection vulnerabilities
- Significant denial-of-service vectors
- Other critical security issues

### What to expect

- We will acknowledge your report within a reasonable timeframe
- We will work to understand and validate the issue
- We will provide updates on our progress and remediation

Thank you for helping keep QueTal Cast and its users safe.
